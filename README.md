## 🚀 Fonctionnalités

- Design responsive et moderne
- Optimisé pour le SEO

## 📦 Installation

```bash
# Cloner le repository
git clone https://github.com/anaskiouaz/portfolio.git

# Ouvrir le fichier HTML dans un navigateur
cd portfolio
open index.html
```
